# ATT Soundpack: Slacker
A World of Warcraft addon. Download locations: [CurseForge](https://www.curseforge.com/wow/addons/att-soundpack-slacker) | [Wago](https://addons.wago.io/addons/att-soundpack-slacker)

[![](https://media.forgecdn.net/attachments/1439/398/buymeacoffee-png.png)](https://www.buymeacoffee.com/Slackluster)

**ATT Soundpack: Slacker** provides multiple soundpacks for All The Things and changes:

- Fanfare sound(s) when collecting new things;
- Complete sound(s) when finishing a category of things;
- Remove sound(s) when a thing is removed from the collection;
- RareFind sound(s) when a treasure or rare is detected;
- Death sound(s) when character death occurs;
- Reporting sound(s) when a report can be made.
